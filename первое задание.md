## ШАГ 7
Создаю EtherChannek между коммутаторами в новосибирске используя интерфейсы G0/1, G0/2 не нарушая требования

На rus-nsk-sw0 (активная сторона):
```
enable
configure terminal
interface range fastEthernet 0/5-6
channel-group 1 mode active
channel-protocol lacp
exit
interface port-channel 1
switchport mode trunk
exit
exit
```

На rus-nsk-sw1 (пассивная сторона):
```
enable
configure terminal
interface range fastEthernet 0/1-2
channel-group 1 mode passive
channel-protocol lacp
exit
interface port-channel 1
switchport mode trunk
exit
exit
```
При настройке LACP одна сторона должна выступать в роли активной , а другая — в роли пассивной. Согласно заданию, коммутатор SW0 (rus-nsk-sw0) является ответственным за инициирование согласования, поэтому он настроен как активный, а SW1 (rus-nsk-sw1) — как пассивный.
